## Installation (WIP)

```bash
paru -S glace-git gtk-session-lock
git clone https://github.com/S4NKALP/Modus
cd Modus
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pip install --no-deps git+https://github.com/Fabric-Development/fabric.git
```
